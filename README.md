# Notes

## One of the projects for my first portfolio (June 2020). A to-do list made with jQuery. This project was a great way to exercise working with arrays and jQuery. :memo:

### Try it on Heroku

https://ado-notes.herokuapp.com/

### Functionalities
* create lists 
* add items 
* mark as complete 
* show/hide completed 
* search 
* delete

### Using
* jQuery
* CSS3
